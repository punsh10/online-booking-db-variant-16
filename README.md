# База данных онлайн-записи в коворкинг. Вариант 16

Выполнил:Олейников М.В.

Группа:454

Дисциплина:Базы данных

Описание предметной области

База данных предназначена для автоматизации онлайн-бронирования рабочих мест в коворкинг-центре. Клиенты могут выбирать и бронировать рабочие места трёх типов: открытое пространство, переговорная комната, отдельный офис. Система поддерживает почасовую оплату, скидки для постоянных клиентов и предотвращает двойное бронирование одного места в одно время.

Стек технологий

- СУБД: MariaDB (MySQL-совместимая)
- Хостинг базы данных: Beget
- Управление БД: phpMyAdmin
- Язык запросов: SQL

Структура репозитория

- `sql/schema.sql` — SQL-скрипт создания всех таблиц базы данных
- `sql/data.sql` — SQL-скрипт с тестовыми данными для заполнения таблиц
- `sql/queries.sql` — SQL-запросы по заданию (JOIN, GROUP BY, аналитика по дням недели)
- `docs/report.md` — полный отчёт по практической работе
- `answer_question_16_variant_16.md` — ответ на контрольный вопрос №16 (индексы)

Как развернуть базу данных

Шаг 1. Клонирование репозитория

Скачайте репозиторий на компьютер:

git clone https://github.com/punsh10/online-booking-db-variant-16.git

Или просто скачайте ZIP-архив через зелёную кнопку Code.

Шаг 2. Импорт схемы

Откройте phpMyAdmin вашего хостинга.

Создайте новую базу данных с именем `kovorking_variant16` и кодировкой `utf8mb4_unicode_ci`.

Перейдите на вкладку SQL и выполните содержимое файла `sql/schema.sql`.

Шаг 3. Загрузка тестовых данных

На вкладке SQL выполните содержимое файла `sql/data.sql`.

Шаг 4. Проверка

Выполните запросы из файла `sql/queries.sql` для проверки работоспособности базы данных.
